# Anythings 🤙

**by TuryTury**

A tool that pulls proxy configs from different sources, organizes them, and checks which ones actually work using Xray.

## Stack

- **Web:** Next.js 15 + Tailwind CSS
- **API:** Fastify + WebSocket
- **Local mode:** Everything in memory, no Docker needed
- **Testing:** Xray Core validates connectivity

## Quick start

- Open powershell and paste this cmd 

```bash
npm install
cp .env.example .env
npm run dev
```

Open `http://localhost:3000`. API runs on port `4000`.
